When do I get this error?

JSHint will throw the "This function has too many parameters" error when it encounters a function signature with more named arguments than specified by the maxparams option. Here's an example in which we attempt to declare a function that takes 3 arguments:

This is the equivalent of the older JSHint "Too many parameters per function" warning. More detail can be found on the page for that message.

In JSHint 1.0.0 and above you have the ability to ignore any warning with a special option syntax. The identifier of this warning is W072. This means you can tell JSHint to not issue this warning with the /*jshint -W072 */ directive.
