When do I get this error?

JSLint will throw the "Bad for in variable '{a}'" error when it encounters an index variable in a for-in loop that is not declared locally. The variable cannot be declared inside the loop or the Move 'var' declarations to the top of the function error will be thrown.

Why do I get this error?

From Crockford himself:

"It wants you to be using a local var. Using a parameter looks like a mistake." Check here.
