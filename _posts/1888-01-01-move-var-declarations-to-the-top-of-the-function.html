Why do I get this error?

This error is raised to highlight a possible misunderstanding of the language. Your code will most likely work as expected if you do not resolve this issue, but you may have misunderstood how JavaScript handles variable declarations.

All declarations are hoisted to the top of the scope in which they appear. Variables in JavaScript have function scope, not block scope. In many other languages, the variable i in the above example would be scopes to the for loop body and would not be accessible to the containing scope. In JavaScript, this is not the case and the variable is actually accessible in the containing scope, although it won't have a value until the loop initialiser has been executed.
